SSO Client JavaScript

This example demonstrates how to integrate with the SSO Gateway from a frontend application.

// Initialize SSO Client
const sso = new SSOClient('http://localhost:3000', 'product1');

// Login
sso.login();

// Logout
sso.logout();

// Get user info
const user = await sso.getUser();

// Check access
const hasAccess = await sso.checkAccess();
